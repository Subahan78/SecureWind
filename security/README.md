# SecureWind

## Secure Wind Turbine Cybersecurity System

SecureWind is a Python-based cybersecurity project that simulates a secure wind turbine management system.

## Features

- User Authentication
- Password Hashing (bcrypt)
- Role-Based Access Control (RBAC)
- Intrusion Detection System (IDS)
- Incident Response System
- AES Encryption
- Security Logging
- Security Dashboard
- Wind Turbine Controller
- Wind Turbine Monitoring
- Log Analyzer

## Technologies Used

- Python
- bcrypt
- cryptography
